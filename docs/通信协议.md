# 通信协议（Frontend <-> C++ Backend）

## 通道名称

- JS 调用入口：`window.nativeInvoke(payload: string): Promise<string>`
- `payload` 为 JSON 字符串。
- 返回值为 JSON 字符串。

## 请求格式

```json
{
  "id": "req-1715243000000",
  "method": "ping",
  "params": {
    "message": "hello from react"
  }
}
```

字段说明：
- `id`：请求唯一标识，字符串。
- `method`：调用方法名，字符串。
- `params`：可选对象参数。

## 响应格式

成功：

```json
{
  "id": "req-1715243000000",
  "result": {
    "echo": "hello from react",
    "method": "ping"
  }
}
```

失败：

```json
{
  "id": "req-1715243000000",
  "error": {
    "code": "METHOD_NOT_FOUND",
    "message": "unsupported method: xxx"
  }
}
```

字段说明：
- `id`：透传请求 ID。
- `result`：成功数据。
- `error`：失败信息，包含 `code` 与 `message`。

## 已实现方法

- `ping`
  - 入参：`params.message?: string`
  - 出参：`{ echo: string, method: "ping" }`
- `getAppInfo`
  - 入参：无
  - 出参：`{ name: string, backend: string, bridgeVersion: string }`
