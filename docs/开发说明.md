# 开发说明

## 目录说明

- `app/frontend`：Vite + React + Tailwind 前端工程。
- `app/backend`：C++17 + webview 宿主工程（CMake，内置本地静态 HTTP 服务）。
- `app/saucer_backend`：C++23 + Saucer 宿主工程（CMake，支持 frontend embedding）。
- `docs/通信协议.md`：前后端 JSON 协议定义。

## 常用命令

安装依赖：

```powershell
pnpm install
```

前端：

```powershell
pnpm frontend:dev
pnpm frontend:build
```

后端构建：

```powershell
pnpm backend:build
pnpm saucer_backend:build
```

## 开发模式（使用前端 dev server）

1) 安装前端依赖

```powershell
pnpm install
```

2) 启动前端开发服务器

```powershell
pnpm --dir app/frontend dev
```

3) 构建 backend 宿主程序（首次或代码更新后）

```powershell
pnpm backend:build
```

4) 启动桌面宿主并加载 dev server

```powershell
$env:WEBVIEW_DEV_SERVER_URL = "http://127.0.0.1:5173"
.\app\backend\build\Release\cpp_webview_host.exe
```

可选：使用 Saucer 宿主加载 dev server

```powershell
pnpm saucer_backend:build
$env:WEBVIEW_DEV_SERVER_URL = "http://127.0.0.1:5173"
$env:WEBVIEW_DEVTOOLS = "1"
.\app\saucer_backend\build\Release\cpp_saucer_host.exe
```

## 生产模式 A：`app/backend`（本地静态 HTTP 服务）

1) 构建前端静态资源

```powershell
pnpm frontend:build
```

2) 关闭 dev server 环境变量并运行宿主

```powershell
Remove-Item Env:WEBVIEW_DEV_SERVER_URL -ErrorAction SilentlyContinue
.\app\backend\build\Release\cpp_webview_host.exe
```

backend 会自动：
- 从 `app/frontend/dist` 提供静态资源服务；
- 使用 `127.0.0.1:0` 动态分配可用端口；
- 导航到 `http://127.0.0.1:<动态端口>/index.html`。

3) 可选：固定端口优先（便于调试）

```powershell
$env:WEBVIEW_STATIC_PORT = "18080"
Remove-Item Env:WEBVIEW_DEV_SERVER_URL -ErrorAction SilentlyContinue
.\app\backend\build\Release\cpp_webview_host.exe
```

如果 `WEBVIEW_STATIC_PORT` 被占用，宿主会自动回退到动态端口。

## 生产模式 B：`app/saucer_backend`（embedded frontend）

前置要求：
- CMake 版本 `>= 3.31`（`app/saucer_backend/CMakeLists.txt` 已声明最低版本）。

1) 构建前端静态资源

```powershell
pnpm frontend:build
```

2) 构建并运行 Saucer 宿主

```powershell
pnpm saucer_backend:build
Remove-Item Env:WEBVIEW_DEV_SERVER_URL -ErrorAction SilentlyContinue
.\app\saucer_backend\build\Release\cpp_saucer_host.exe
```

Saucer 模式会将 `dist` 资源嵌入并通过内置 scheme 提供，不依赖本地 HTTP 端口。

如果当前终端 `cmake` 版本低于 3.31，可在当前会话临时优先使用 VS CMake：

```powershell
$env:Path = "C:\Program Files (x86)\Microsoft Visual Studio\2022\BuildTools\Common7\IDE\CommonExtensions\Microsoft\CMake\CMake\bin;$env:Path"
pnpm saucer_backend:build
```
